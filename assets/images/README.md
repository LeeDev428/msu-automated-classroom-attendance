# App Logo and Images

Place your app logo and images in this folder.

## Required Files:
- **Automated-Classroom-Attendance-Logo.png** - Main app logo/icon

## Usage:
The logo can be used in the screens by importing it:
```javascript
import logo from '../../assets/images/Automated-Classroom-Attendance-Logo.png';
```

Then display it using:
```javascript
<Image source={logo} style={styles.logo} />
```
